01. Semantic Code Refactorer

AI / Full-Stack Application

Timeline: May 2025 - June 2025

Goal & Impact

to automate intelligent, context-aware code improvements that go beyond simple syntax checking. this tool analyzes abstract semantics to suggest and execute complex transformations like renaming, restructuring, and simplification, vastly improving code quality and maintainability.

Architecture

  • Frontend: Built with React to provide a seamless interface for inputting code, selecting refactoring types, and viewing the changes.
  • Backend: A robust FastAPI service handles concurrent requests and manages the application logic accompanied with PostgreSQL for data persistence.
  • AI Core: Leveraged the Gemini API with advanced prompt engineering techniques for context-aware code pattern detection and semantic transformation generation. Uses PostgreSQL for data persistence.

Technologies

Gemini API FastAPI React PostgreSQL

02. Multi-modal Cancer Grading Framework

Deep Learning / Medical Vision Research

Timeline: May 2025 - June 2025

Goal & Impact

developed a deep learning solution under the MMVL Lab to improve the accuracy and reliability of cancer grading by integrating multiple data modalities, based on the principles of the "IHC Matters" research paper.

Methodology

  • Model: Implemented an Attention-based Multiple Instance Learning (MIL) approach to handle whole-slide imaging data effectively.
  • Innovation: Featured a two-stage pipeline incorporating advanced techniques like feature fusion and Generative Models for enhanced representation learning.
  • Scale: Trained and validated the framework on a dataset comprising over 5,000 medical images.

03. TSPN PyTorch Implementation

Research Module / Transformers

Timeline: November 2025

Goal & Impact

to create a high-fidelity, from-scratch implementation of the Transformer-based Set Prediction Network (TSPN). this serves as a foundational and flexible module for integrating set prediction capabilities into larger research-focused deep learning projects.

Implementation Details

  • Core: Built the entire network from fundamental components in PyTorch, avoiding high-level wrappers. This approach ensures maximum control over layer implementation, optimization, and debugging. [Image of Transformer Architecture]
  • Application: Designed specifically to be used as a replaceable module within complex research architectures, emphasizing clear code structure and high performance on scientific datasets.
